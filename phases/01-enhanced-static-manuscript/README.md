# Phase 1: Enhanced Static Manuscript

Goal: turn the Markdown chapters into a readable static Web manuscript.

This phase owns:

- the static site generator;
- manuscript typography and layout assets;
- styled exhibit placeholders;
- styled Wizard's rule cards;
- stable anchors for notes and references;
- reader settings for font, size, line height, column width, theme and motion.

The generated output is written to `site/`.
